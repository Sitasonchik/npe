# NPE
> Not for prying eyes

## About NPE
This prog made for file encryption such as documents on C++. 
Cryprography based on openssl. AES was used for encryption.

## Dependencies
This prog required openssl, cmake.
Command to install on arch: ```sudo pacman -S cmake openssl```

## How to use 
1) launch app
2) choose what will be done with file(decrypt or encrypt)
3) write file name, which will be rewrited by encrypted text from this file
4) write the key to encrypt or decrypt
